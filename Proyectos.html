Validacion Tarjeta

Este proyecto consiste en la elaboración de una página web que sea capaz de validar una tarjeta de crédito al ingresar dicho número mediante el algoritmo de Luhn.Dicho algoritmo también es conocido como algoritmo base 10, que hoy en día se utiliza para verificar tarjetas de crédito.

Calorias App

Este proyecto consiste en una página web que calcula las calorías de diferentes tipos de comida rápida como pizza, hamburguesas, helados, etc.Luego genera la suma de calorías totales consumidas si se seleccionan cierta cantidad de comidas.

Trivia Kodigo

Consiste en una página web que contiene dos cuestionarios sobre ciertos temas, y por cada cuestionario las preguntas son dinámicas y diferentes. Mientras se va avanzando en el juego la página va dando los resultados reales en cada partida.
